# AI 게시글 자동 생성 기능

## 개요
DART 공시와 네이버 뉴스를 교차 분석하여 주식 시장 분석 게시글을 자동 생성한다.

## 크롤링 데이터 현황과 개선

### 현재 수집하는 데이터
| 소스 | 수집 내용 | 한계 |
|------|----------|------|
| DART `/api/list.json` | 공시 목록 (회사명, 보고서명, 접수일) | 제목뿐, 실제 재무 수치 없음 |
| 네이버 뉴스 API | 뉴스 제목 + description 스니펫 | 본문 없이 요약 2~3줄만 |

### 추가 수집이 필요한 데이터
| 소스 | 엔드포인트 | 수집 내용 | 왜 필요한지 |
|------|-----------|----------|------------|
| DART 주요계정 | `/api/fnlttSinglAcnt.json` | 매출, 영업이익, 순이익, EPS, BPS, 부채비율 | 실적 분석의 핵심 수치 |
| DART 주요계정 (전기) | 위와 동일, `bsns_year` 변경 | 전년 동기 실적 | 전년비 증감률 계산 |

### 개선 후 AI에 들어가는 인풋 예시

```
## DART 공시 — 삼성전자 (005930)

### 공시 정보
- 보고서: 분기보고서 (2026년 1분기)
- 접수일: 2026-05-15
- DART 링크: https://dart.fss.or.kr/dsaf001/main.do?rcpNo=20260515001234

### 재무 수치 (단위: 백만원)
| 항목 | 2026 1Q | 2025 1Q | 증감률 |
|------|---------|---------|--------|
| 매출액 | 78,000,000 | 71,000,000 | +9.8% |
| 영업이익 | 12,500,000 | 9,800,000 | +27.5% |
| 순이익 | 9,200,000 | 7,100,000 | +29.6% |
| EPS | 6,150원 | 4,750원 | +29.5% |

### 관련 뉴스 (최근 7일)
1. "삼성전자, HBM3E 양산 본격화...반도체 업황 회복 신호" (2026-05-14)
2. "외국인 삼성전자 5거래일 연속 순매수" (2026-05-13)
3. "반도체 수출 전년비 45% 증가...코스피 상승 견인" (2026-05-12)

### 과거 유사 데이터 (벡터 DB 검색)
- [2025-11-15] 삼성전자 3분기 실적 공시 — 영업이익 9.8조, 전년비 +15%
  → 당시 뉴스: "AI 반도체 수요 증가 전망"
- [2025-05-16] 삼성전자 1분기 실적 공시 — 영업이익 6.5조, 전년비 +55%
  → 당시 뉴스: "반도체 바닥론 확산"
```

## 분석 데이터와 역할

### DART 공시 (팩트)
| 공시유형 | 데이터 | 주가 영향 |
|---------|--------|----------|
| 주요사항보고(B) | 유상증자, 자사주 매입, 대규모 계약, 합병 | 단기 급등/급락 직접 원인 |
| 정기공시(A) | 사업보고서, 분기보고서 (매출, 영업이익, EPS) | 실적 서프라이즈/쇼크 판단 기준 |
| 지분공시(D) | 대량보유 보고, 임원 지분 변동 | 기관/외국인 수급 신호 |
| **주요계정 조회** | **매출, 영업이익, 순이익, EPS, BPS, 부채비율** | **전년비 증감률로 실적 방향 판단** |

### 네이버 뉴스 (센티먼트)
| 키워드 | 데이터 | 분석 용도 |
|--------|--------|----------|
| 주식, 코스피 | 지수 동향, 시장 전반 분위기 | 전체 시장 방향성 |
| 경제, 금리 | 금리 인상/인하, 환율, 정책 | 매크로 환경 판단 |
| 종목명 | 특정 기업 관련 뉴스 | 공시와 교차 검증 |

## 흐름

```
[스케줄러 or 수동 트리거]
        │
        ▼
1. 벡터 DB에서 최근 크롤링 문서 조회
   - source="dart" → 공시 목록
   - source="dart-financial" → 재무 수치
   - source="naver-news" → 뉴스
        │
        ▼
2. 종목/키워드 기준으로 연관 데이터 벡터 검색
        │
        ▼
3. 프롬프트 조립
        │
        ▼
4. ChatModel 호출 → 분석 게시글 생성
        │
        ▼
5. Board 모듈로 게시글 등록
```

## 고려 사항

### 비용
- GPT-4o-mini 기준, 인풋 ~3000토큰 + output ~1500토큰
- 1회 약 $0.002, 하루 5~10건 시 월 $1~2

### 트리거 방식
- 스케줄러: 크롤링 완료 후 자동 실행 (예: 매일 오전 9시)
- 수동: `POST /ai/posts/generate` 엔드포인트
